# 📦 Sistema de Análise de Estoque

Este projeto foi desenvolvido com foco em aprendizado prático, aplicando conceitos estudados na faculdade junto com situações reais vivenciadas no ambiente de trabalho.

O objetivo principal não é criar uma solução completa ou comercial, mas sim entender melhor o funcionamento de um sistema de controle e análise de estoque na prática.

## 🎯 Objetivo

* Praticar lógica de programação em Python
* Aplicar conceitos acadêmicos em um contexto real
* Simular operações básicas de controle de estoque
* Compreender o fluxo de entrada, saída e análise de dados

## 🚀 Funcionalidades implementadas

* Cadastro de produtos
* Controle de entrada e saída de estoque
* Geração de relatórios em Excel
* Organização básica dos dados

## 🛠️ Tecnologias utilizadas

* Python
* Pandas
* Excel

## 📁 Estrutura do projeto

* `estoque.py` → lógica de controle de estoque
* `relatorio.py` → geração de relatórios
* `ativar_isto.py` → execução principal do sistema
* `dashboard_estoque.pbix` → visualização no Power BI

## ▶️ Como executar

1. Instale as dependências:

```
pip install pandas openpyxl
```

2. Execute o sistema:

```
python ativar_isto.py
```

## 📊 Observação

Este é um projeto simples, desenvolvido com fins educacionais, como forma de consolidar conhecimentos em programação e análise de dados.
